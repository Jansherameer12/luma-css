Core Design System

Define your palette, geometry, and motion with CSS variables.

Colors

:root {
  --luma-prime: #5E5BFF;
  --luma-ink: #1A1A1A;
  --luma-paper: #FFFFFF;
}

Geometry

:root {
  --luma-edge: 0.5rem;
  --luma-edge-sm: 0.25rem;
  --luma-edge-lg: 1rem;
}

Motion

:root {
  --luma-move: 0.25s cubic-bezier(0.4,0,0.2,1);
  --luma-bounce: 0.5s cubic-bezier(0.68,-0.55,0.265,1.55);
}

</p>

</p>

  UPPERCASE TEXT
</p>

  Capitalized Text
</p>

Font Weight

  Regular text
</p>
<p class="luma-tx-600">
  Semi-bold text
</p>
